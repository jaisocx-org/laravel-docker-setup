# Symfony under Jaisocx webserver docker image

1. git clone git@github.com:jaisocx-org/symfony-docker-setup.git
2. docker-compose build
3. docker-compose up
4. docker-compose exec php composer install
4. http://localhost:2291/
4. That's all!
